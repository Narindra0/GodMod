# GODMOD v2

Système intelligent de prédiction pour matchs virtuels de football.

## Documentation

- `docs/GODMOD.md`
- `docs/ZEUS.md`

## Structure du projet (résumé)

- `src/` : code applicatif (core, scrapers, analysis, zeus)
- `scripts/` : scripts utilitaires (migration DB, tests ponctuels)
- `data/` : base SQLite + archives (données)
- `models/` : modèles IA (poids / checkpoints)
- `logs/` : journaux d’exécution (générés)
